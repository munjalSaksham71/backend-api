Documentation

Test this API on Postman

It is a two way API : Seller And Buyer

Seller's Routes :

GET and POST Route to get products and add new Product
https://crud-api-25f21.herokuapp.com/api/seller

GET product by id, DELETE and UPDATE Route to get product by id, Delete Product and Update Product resp
https://crud-api-25f21.herokuapp.com/api/seller/{id}

Buyer's Routes :

Add To WishList

GET and POST Route to get All the products in wishList and add new Product in wishList
https://crud-api-25f21.herokuapp.com/api/buyer/wishList

DELETE and UPDATE Route to remove product from wishList or to update the quanity in cart resp
https://crud-api-25f21.herokuapp.com/api/buyer/wishList/{id}

Orders:

GET and POST Route to get All the Orders and New Orders resp
https://crud-api-25f21.herokuapp.com/api/buyer/order

GET order By id, DELETE and UPDATE Route to get order info by id, cancel order and to update the order.
https://crud-api-25f21.herokuapp.com/api/buyer/order/{id}

NOTE: Make Sure while POST request Your Content-Type must be Application/json in raw format.
